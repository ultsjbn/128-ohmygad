# OhMyGAD!
## Event Mangement Platform for UPB Kasarian Gender Studies Program
Developers:
Abiva, De Guzman, Novesteras, Solis

---
## Overview
OhMyGAD! is a web-based event management platform designed specifically for the University of the Philippines Baguio (UPB) Kasarian Gender Studies Program. It centralizes event management, course catalogs, demographic analytics, and post-event surveys with full role-based access control for students, faculty, and administrators.
